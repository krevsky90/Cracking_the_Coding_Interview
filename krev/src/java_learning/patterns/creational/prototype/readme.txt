идея:
если есть группа иерархие классов, нам НЕ нужна иерархия фабрик с фабричным методом, т.к. процесс создания нового объекта каждого класса зашит в самом классе!

Алгоритм:
1) Создать интерфейс/абстрактный класс с абстрактным методом clone()
2) отнаследоваться неким классом (скажем, MyObject)
3) в MyObject:
    - создать обычный конструктор для создания объекта и инициализации полей
    - создать конструктор MyObject(MyObject obj), создающий объект на основе содержимого переданного obj
    - имплементировать метод clone() { return MyObject(this); }
