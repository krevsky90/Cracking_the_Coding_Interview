idea:
снепшот состояния объекта делает САМ объект, т.к. объект имеет доступ ко всем своим полям
восстановление состояния из снепшота делает
    либо сам объект:
        для этого снепшот должен
    либо сам снепшот (имея ссылку на объект, к-ый надо восстановить). это лучше

1) inner
2) outer
3) self_restored_snapshot
4) self_restored_snapshot_multiple
5) memento_command (separate folder at 1 level above!)


Применимость:
1) Когда вам нужно сохранять мгновенные снимки состояния объекта (или его части),
    чтобы впоследствии объект можно было восстановить в том же состоянии.

2) Когда прямое получение состояния объекта раскрывает приватные детали его реализации, нарушая инкапсуляцию.
